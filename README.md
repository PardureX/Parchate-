# Parchate™ — Reproductor MP3

**Reproductor de música y radio en vivo con ecualizador y visualizador de audio.**

---

## 🚀 Características

| Funcionalidad | Descripción |
|---------------|-------------|
| 🎧 **Reproductor local** | Arrastra archivos MP3, WAV, OGG, M4A |
| 📻 **Radio en vivo** | Miles de estaciones de radio por internet |
| ⭐ **Emisoras favoritas** | Guarda tus estaciones favoritas (localStorage) |
| 🎚️ **Ecualizador** | Graves, medios, agudos con presets |
| 🔊 **Amplificador** | Boost de volumen hasta 4x |
| 💀 **Visualizador** | Calavera animada con notas musicales |
| 🌓 **Tema oscuro/claro** | Cambia con un clic |
| 🎨 **Colores personalizables** | 6 colores de acento |
| 📱 **PWA** | Funciona offline, instalable en celular |
| ⌨️ **Atajos de teclado** | Espacio (play/pause), flechas (anterior/siguiente) |

---

## 🔍 Búsqueda de radio

Escribe cualquier género, artista o país en el buscador:
cumbia, vallenato, rock, jazz, salsa, reggae, techno, classical, hip hop, mexico, colombia...

text

La búsqueda usa [Radio Browser API](https://www.radio-browser.info/) — una base de datos comunitaria con miles de estaciones de radio por internet.

---

## 🛠️ Tecnologías

- **HTML5 + CSS3 + JavaScript** (vanilla, sin frameworks)
- **Web Audio API** — ecualizador y cadena de audio
- **Canvas API** — visualizador de calavera animada
- **MediaSession API** — controles en pantalla de bloqueo
- **Radio Browser API** — búsqueda de estaciones
- **localStorage** — persistencia de configuración y favoritos
